# 🎨 CV Web | Paola de Fátima Gaona Nolasco

Diseñadora multimedia enfocada en ilustración, diseño digital y creación de contenido visual.

Este proyecto consiste en un **Curriculum Vitae interactivo** desarrollado con tecnologías web, acompañado de una sección de portafolios visuales.

---

## Vista previa CV

![Preview](https://raw.githubusercontent.com/Misho120/Des-app-web/main/02-maquetacion-cv/img/preview.jpg)
---

## ✨ Características

- Diseño visual moderno y limpio  
- Animaciones CSS (título dinámico y efectos visuales)  
- Sección de portafolios organizada por categorías  
- Navegación entre proyectos  
- Integración de imágenes y video como evidencia  

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- Diseño responsive básico  
- Animaciones con keyframes  

---

## 🎯 Contenido del portafolio

- 🎨 Ilustración digital  
- 🧾 Carteles y publicidad  
- 📖 Comics e historietas  
- 🧩 Diseño de logotipos  
- 🎥 Modelado 3D y edición de video  

---

## 🚀 Cómo visualizar el proyecto

1. Descargar o clonar el repositorio  
2. Abrir el archivo `index.html` en el navegador
